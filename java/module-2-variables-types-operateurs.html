Types Primitifs Java

Java possède 8 types primitifs qui stockent directement les valeurs en mémoire (pas d'objets). Ces types sont optimisés pour les performances.

Type Taille Plage de Valeurs Valeur par Défaut Exemple
byte 8 bits -128 à 127 0 byte age = 25;
short 16 bits -32,768 à 32,767 0 short annee = 2024;
int 32 bits -2^31 à 2^31-1 0 int population = 67000000;
long 64 bits -2^63 à 2^63-1 0L long distance = 384400000L;
float 32 bits ±3.4e38 (7 décimales) 0.0f float prix = 19.99f;
double 64 bits ±1.7e308 (15 décimales) 0.0d double pi = 3.141592653589793;
char 16 bits 0 à 65,535 (Unicode) '\u0000' char lettre = 'A';
boolean 1 bit true ou false false boolean actif = true;
Points Importants :
  • int est le type par défaut pour les entiers
  • double est le type par défaut pour les décimaux
  • long nécessite le suffixe 'L' : long x = 123L;
  • float nécessite le suffixe 'f' : float x = 12.3f;
  • char utilise des guillemets simples : 'A'

Exemple Pratique avec Tous les Types

public class TypesPrimitifs {
    public static void main(String[] args) {
        // Types entiers
        byte nombrePetit = 100;
        short nombreMoyen = 30000;
        int nombreStandard = 2147483647;
        long nombreGrand = 9223372036854775807L;
        
        // Types décimaux
        float prixFloat = 29.99f;
        double prixDouble = 299.99;
        
        // Caractère et booléen
        char initiale = 'J';
        boolean estValide = true;
        
        // Affichage des valeurs
        System.out.println("=== DÉMONSTRATION DES TYPES PRIMITIFS ===");
        System.out.println("Byte: " + nombrePetit);
        System.out.println("Short: " + nombreMoyen);
        System.out.println("Int: " + nombreStandard);
        System.out.println("Long: " + nombreGrand);
        System.out.println("Float: " + prixFloat);
        System.out.println("Double: " + prixDouble);
        System.out.println("Char: " + initiale);
        System.out.println("Boolean: " + estValide);
        
        // Affichage des tailles en mémoire
        System.out.println("\n=== TAILLES EN MÉMOIRE ===");
        System.out.println("Byte: " + Byte.BYTES + " octets");
        System.out.println("Short: " + Short.BYTES + " octets");
        System.out.println("Int: " + Integer.BYTES + " octets");
        System.out.println("Long: " + Long.BYTES + " octets");
        System.out.println("Float: " + Float.BYTES + " octets");
        System.out.println("Double: " + Double.BYTES + " octets");
        System.out.println("Char: " + Character.BYTES + " octets");
    }
}

Types Références

Les types références stockent l'adresse mémoire d'un objet, pas la valeur directement. Ils incluent les classes, interfaces, tableaux et énumérations.

Différences Types Primitifs vs Références

Types Primitifs
  • Stockent la valeur directement
  • Stockés dans la pile (stack)
  • Comparaison avec ==
  • Pas de méthodes
  • Performance optimale
Types Références
  • Stockent l'adresse de l'objet
  • Objets dans le tas (heap)
  • Comparaison avec .equals()
  • Possèdent des méthodes
  • Valeur par défaut : null

Exemple avec String (Type Référence)

public class TypesReferences {
    public static void main(String[] args) {
        // Déclaration de chaînes de caractères
        String nom = "Java";
        String langage = new String("Java");
        String autreNom = "Java";
        
        // Comparaison d'adresses (==)
        System.out.println("nom == langage: " + (nom == langage)); // false
        System.out.println("nom == autreNom: " + (nom == autreNom)); // true (String pool)
        
        // Comparaison de contenu (.equals())
        System.out.println("nom.equals(langage): " + nom.equals(langage)); // true
        System.out.println("nom.equals(autreNom): " + nom.equals(autreNom)); // true
        
        // Méthodes disponibles sur String
        System.out.println("Longueur: " + nom.length());
        System.out.println("Majuscules: " + nom.toUpperCase());
        System.out.println("Contient 'av': " + nom.contains("av"));
        
        // Valeur null
        String vide = null;
        System.out.println("Chaîne vide: " + vide);
        // System.out.println(vide.length()); // NullPointerException !
    }
}

│ autreNom ────────────▶│ "Java"      │ (même objet que nom)

Déclaration et Initialisation des Variables

Syntaxe de Déclaration

// Syntaxe générale : type nomVariable [= valeur];

// Déclaration simple
int age;
String nom;

// Déclaration avec initialisation
int age = 25;
String nom = "Alice";

// Déclarations multiples du même type
int x = 10, y = 20, z = 30;
String prenom = "Jean", ville = "Paris";

// Déclaration de constantes
final double PI = 3.14159;
final int NOMBRE_JOURS_SEMAINE = 7;
Règles de Nommage des Variables :
  • Commence par : lettre, $ ou _ (pas de chiffre)
  • Contient : lettres, chiffres, $ et _
  • Sensible à la casse : age ≠ Age ≠ AGE
  • Pas de mots-clés : int, class, public, etc.
  • Convention : camelCase pour variables et méthodes

Exemples de Nommage

// ✅ Bons noms de variables
int ageUtilisateur = 30;
String nomComplet = "Jean Dupont";
double salaireAnnuel = 45000.0;
boolean estConnecte = false;
char premiereLettre = 'J';

// ❌ Mauvais noms de variables
int a = 30;                    // Pas descriptif
String n = "Jean";             // Trop court
double sal$ = 45000.0;         // Symboles non standards
boolean b1 = false;            // Pas clair
int 2age = 30;                 // Commence par un chiffre (erreur)
String class = "MaClasse";     // Mot-clé réservé (erreur)

Opérateurs d'Incrémentation et Décrémentation

public class OperateursIncrements {
    public static void main(String[] args) {
        int compteur = 10;
        
        // Pré-incrémentation (incrémente puis utilise)
        System.out.println("Compteur initial: " + compteur);        // 10
        System.out.println("Pré-incrémentation: " + (++compteur));  // 11
        System.out.println("Après pré-incrémentation: " + compteur); // 11
        
        // Post-incrémentation (utilise puis incrémente)
        compteur = 10; // Reset
        System.out.println("Post-incrémentation: " + (compteur++)); // 10
        System.out.println("Après post-incrémentation: " + compteur); // 11
        
        // Même principe pour la décrémentation
        System.out.println("Pré-décrémentation: " + (--compteur));  // 10
        System.out.println("Post-décrémentation: " + (compteur--)); // 10
        System.out.println("Valeur finale: " + compteur);           // 9
        
        // Utilisation pratique dans une boucle
        System.out.println("\nComptage de 1 à 5:");
        for (int i = 1; i <= 5; i++) {
            System.out.print(i + " ");
        }
    }
}

Les opérateurs de comparaison retournent toujours une valeur booléenne (true ou false).

Opérateur Description Exemple Résultat
==Égal à5 == 5true
!=Différent de5 != 3true
>Supérieur à5 > 3true
<Inférieur à5 < 3false
>=Supérieur ou égal5 >= 5true
<=Inférieur ou égal3 <= 5true
Attention avec les Types Références !

Pour les objets (String, etc.), utilisez .equals() pour comparer le contenu :

String a = new String("Java");
String b = new String("Java");

System.out.println(a == b);        // false (adresses différentes)
System.out.println(a.equals(b));   // true (contenu identique)

Opérateurs Logiques

Les opérateurs logiques permettent de combiner des expressions booléennes.

Opérateurs Principaux

Opérateur Description Exemple
&&ET logiquetrue && false = false
||OU logiquetrue || false = true
!NON logique!true = false
&ET (sans court-circuit)true & false = false
|OU (sans court-circuit)true | false = true

Table de Vérité

A B A && B A || B !A
truetruetruetruefalse
truefalsefalsetruefalse
falsetruefalsetruetrue
falsefalsefalsefalsetrue

Exemple Pratique - Validation d'Âge

public class OperateursLogiques {
    public static void main(String[] args) {
        int age = 20;
        boolean aPermisConduire = true;
        boolean aVoiture = false;
        
        // Conditions composées
        boolean peutConduire = (age >= 18) && aPermisConduire;
        boolean peutConduireSeul = peutConduire && aVoiture;
        boolean besoinFormation = (age < 25) || !aPermisConduire;
        
        System.out.println("Age: " + age);
        System.out.println("A un permis: " + aPermisConduire);
        System.out.println("A une voiture: " + aVoiture);
        System.out.println();
        
        System.out.println("Peut conduire: " + peutConduire);
        System.out.println("Peut conduire seul: " + peutConduireSeul);
        System.out.println("Besoin de formation: " + besoinFormation);
        
        // Court-circuit avec &&
        if (aVoiture && verifierAssurance()) {
            System.out.println("Voiture assurée et disponible");
        }
        
        // Court-circuit avec ||
        if (age >= 65 || aHandicap()) {
            System.out.println("Tarif réduit applicable");
        }
    }
    
    public static boolean verifierAssurance() {
        System.out.println("Vérification assurance...");
        return true;
    }
    
    public static boolean aHandicap() {
        System.out.println("Vérification handicap...");
        return false;
    }
}

Conversion de Types (Casting)

Conversion Implicite (Widening)

Java effectue automatiquement les conversions qui ne risquent pas de perte de données :

Ordre de Conversion Automatique :

byte → short → int → long → float → double

char → int → long → float → double

public class ConversionImplicite {
    public static void main(String[] args) {
        // Conversions automatiques (élargissement)
        byte petitNombre = 100;
        short moyenNombre = petitNombre;    // byte → short
        int grandNombre = moyenNombre;      // short → int
        long tresGrandNombre = grandNombre; // int → long
        float nombreDecimal = tresGrandNombre; // long → float
        double precisionDouble = nombreDecimal; // float → double
        
        System.out.println("Byte: " + petitNombre);
        System.out.println("Short: " + moyenNombre);
        System.out.println("Int: " + grandNombre);
        System.out.println("Long: " + tresGrandNombre);
        System.out.println("Float: " + nombreDecimal);
        System.out.println("Double: " + precisionDouble);
        
        // Conversion char → int
        char lettre = 'A';
        int codeAscii = lettre;
        System.out.println("Lettre '" + lettre + "' = " + codeAscii);
    }
}

Conversion Explicite (Narrowing)

Pour les conversions risquant une perte de données, le casting explicite est obligatoire :

public class ConversionExplicite {
    public static void main(String[] args) {
        // Conversions explicites (rétrécissement)
        double prixDouble = 19.99;
        float prixFloat = (float) prixDouble;    // double → float
        long prixLong = (long) prixFloat;        // float → long
        int prixInt = (int) prixLong;            // long → int
        short prixShort = (short) prixInt;       // int → short
        byte prixByte = (byte) prixShort;        // short → byte
        
        System.out.println("Double: " + prixDouble);
        System.out.println("Float: " + prixFloat);
        System.out.println("Long: " + prixLong);
        System.out.println("Int: " + prixInt);
        System.out.println("Short: " + prixShort);
        System.out.println("Byte: " + prixByte);
        
        // Attention aux débordements !
        int grandNombre = 300;
        byte petitNombre = (byte) grandNombre;
        System.out.println("300 en byte: " + petitNombre); // 44 (débordement !)
        
        // Conversion String ↔ nombres
        String texteNombre = "123";
        int nombre = Integer.parseInt(texteNombre);
        String retourTexte = String.valueOf(nombre);
        
        System.out.println("String vers int: " + nombre);
        System.out.println("Int vers String: " + retourTexte);
    }
}

Exercice 4 : Calculateur d'IMC (Indice de Masse Corporelle)

Objectif : Créer un calculateur d'IMC utilisant différents types et opérateurs.

Formule : IMC = poids (kg) / (taille (m))²

Fonctionnalités :

  • Saisie du poids et de la taille
  • Calcul de l'IMC avec précision
  • Interprétation du résultat
  • Gestion des erreurs de saisie
Solution Complète :
import java.util.Scanner;

public class CalculateurIMC {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("=== CALCULATEUR D'IMC ===");
        System.out.println("Indice de Masse Corporelle");
        System.out.println();
        
        try {
            // Saisie des données
            System.out.print("Entrez votre poids (kg) : ");
            double poids = scanner.nextDouble();
            
            System.out.print("Entrez votre taille (m) : ");
            double taille = scanner.nextDouble();
            
            // Validation des entrées
            if (poids <= 0 || taille <= 0) {
                System.out.println("Erreur: Le poids et la taille doivent être positifs");
                return;
            }
            
            if (poids > 500 || taille > 3.0) {
                System.out.println("Erreur: Valeurs non réalistes");
                return;
            }
            
            // Calcul de l'IMC
            double imc = poids / (taille * taille);
            
            // Affichage des résultats
            System.out.println();
            System.out.println("=== RÉSULTATS ===");
            System.out.printf("Poids: %.1f kg%n", poids);
            System.out.printf("Taille: %.2f m%n", taille);
            System.out.printf("IMC: %.2f%n", imc);
            System.out.println();
            
            // Interprétation
            String interpretation = interpreterIMC(imc);
            System.out.println("Interprétation: " + interpretation);
            
            // Conseils
            donnerConseils(imc);
            
        } catch (Exception e) {
            System.out.println("Erreur de saisie: Veuillez entrer des nombres valides");
        } finally {
            scanner.close();
        }
    }
    
    public static String interpreterIMC(double imc) {
        if (imc < 18.5) {
            return "Insuffisance pondérale";
        } else if (imc < 25) {
            return "Poids normal";
        } else if (imc < 30) {
            return "Surpoids";
        } else if (imc < 35) {
            return "Obésité modérée";
        } else if (imc < 40) {
            return "Obésité sévère";
        } else {
            return "Obésité morbide";
        }
    }
    
    public static void donnerConseils(double imc) {
        System.out.println("=== CONSEILS ===");
        
        if (imc < 18.5) {
            System.out.println("Consultez un professionnel de santé pour prendre du poids sainement.");
        } else if (imc < 25) {
            System.out.println("Félicitations ! Votre poids est dans la norme.");
            System.out.println("Maintenez une alimentation équilibrée et une activité physique régulière.");
        } else if (imc < 30) {
            System.out.println("Attention au surpoids. Considérez :");
            System.out.println("- Une alimentation plus équilibrée");
            System.out.println("- Une activité physique régulière");
        } else {
            System.out.println("Il est recommandé de consulter un professionnel de santé.");
            System.out.println("Un suivi médical peut être nécessaire.");
        }
    }
}
Exercice 5 : Système de Monnaie

Objectif : Créer un système qui calcule le rendu de monnaie optimal.

Spécifications :

  • Prix d'un article et montant payé
  • Calcul du rendu en billets et pièces
  • Optimisation du nombre de pièces/billets
  • Gestion des centimes avec double
Solution Avancée :
import java.util.Scanner;

public class SystemeMonnaie {
    // Constantes pour les valeurs de monnaie (en centimes)
    private static final int[] VALEURS = {
        50000, 20000, 10000, 5000, 2000, 1000, 500, 200, 100, 50, 20, 10, 5, 2, 1
    };
    
    private static final String[] NOMS = {
        "500€", "200€", "100€", "50€", "20€", "10€", "5€", "2€", "1€", 
        "50c", "20c", "10c", "5c", "2c", "1c"
    };
    
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("=== SYSTÈME DE RENDU DE MONNAIE ===");
        System.out.println();
        
        try {
            System.out.print("Prix de l'article (€) : ");
            double prix = scanner.nextDouble();
            
            System.out.print("Montant payé (€) : ");
            double montantPaye = scanner.nextDouble();
            
            if (montantPaye < prix) {
                System.out.println("Erreur: Le montant payé est insuffisant");
                return;
            }
            
            double rendu = montantPaye - prix;
            
            if (rendu == 0) {
                System.out.println("Montant exact, pas de rendu à effectuer");
                return;
            }
            
            System.out.println();
            System.out.printf("Prix: %.2f €%n", prix);
            System.out.printf("Payé: %.2f €%n", montantPaye);
            System.out.printf("Rendu: %.2f €%n", rendu);
            System.out.println();
            
            // Conversion en centimes pour éviter les erreurs de précision
            int renduCentimes = (int) Math.round(rendu * 100);
            
            System.out.println("=== DÉTAIL DU RENDU ===");
            calculerRendu(renduCentimes);
            
        } catch (Exception e) {
            System.out.println("Erreur de saisie: " + e.getMessage());
        } finally {
            scanner.close();
        }
    }
    
    public static void calculerRendu(int renduCentimes) {
        int totalBilletsPieces = 0;
        
        for (int i = 0; i < VALEURS.length; i++) {
            int quantite = renduCentimes / VALEURS[i];
            
            if (quantite > 0) {
                System.out.println(quantite + " x " + NOMS[i] + 
                                 " = " + String.format("%.2f €", (quantite * VALEURS[i]) / 100.0));
                renduCentimes -= quantite * VALEURS[i];
                totalBilletsPieces += quantite;
            }
        }
        
        System.out.println();
        System.out.println("Total: " + totalBilletsPieces + " billets/pièces");
        
        if (renduCentimes > 0) {
            System.out.println("Reste non rendu: " + renduCentimes + " centimes");
        }
    }
}
Exercice 6 : Jeu de Devinette Numérique

Objectif : Créer un jeu qui utilise opérateurs de comparaison et logiques.

Règles :

  • L'ordinateur génère un nombre aléatoire entre 1 et 100
  • L'utilisateur doit deviner en maximum 7 tentatives
  • Indices : "trop grand", "trop petit"
  • Score basé sur le nombre de tentatives
Solution Complète :
import java.util.Scanner;
import java.util.Random;

public class JeuDevinette {
    private static final int NOMBRE_MIN = 1;
    private static final int NOMBRE_MAX = 100;
    private static final int MAX_TENTATIVES = 7;
    
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        Random random = new Random();
        
        System.out.println("=== JEU DE DEVINETTE NUMÉRIQUE ===");
        System.out.println("Devinez le nombre entre " + NOMBRE_MIN + " et " + NOMBRE_MAX);
        System.out.println("Vous avez " + MAX_TENTATIVES + " tentatives maximum");
        System.out.println();
        
        int nombreSecret = random.nextInt(NOMBRE_MAX - NOMBRE_MIN + 1) + NOMBRE_MIN;
        int tentatives = 0;
        boolean trouve = false;
        
        while (tentatives < MAX_TENTATIVES && !trouve) {
            tentatives++;
            System.out.print("Tentative " + tentatives + "/" + MAX_TENTATIVES + " : ");
            
            try {
                int proposition = scanner.nextInt();
                
                // Validation de la proposition
                if (proposition < NOMBRE_MIN || proposition > NOMBRE_MAX) {
                    System.out.println("Le nombre doit être entre " + NOMBRE_MIN + " et " + NOMBRE_MAX);
                    tentatives--; // Ne pas compter cette tentative
                    continue;
                }
                
                // Comparaison avec le nombre secret
                if (proposition == nombreSecret) {
                    trouve = true;
                    System.out.println("🎉 BRAVO ! Vous avez trouvé le nombre " + nombreSecret + " !");
                    calculerScore(tentatives);
                } else if (proposition < nombreSecret) {
                    System.out.println("Trop petit !");
                    donnerIndice(proposition, nombreSecret, tentatives);
                } else {
                    System.out.println("Trop grand !");
                    donnerIndice(proposition, nombreSecret, tentatives);
                }
                
            } catch (Exception e) {
                System.out.println("Veuillez entrer un nombre valide");
                tentatives--; // Ne pas compter cette tentative
                scanner.nextLine(); // Vider le buffer
            }
        }
        
        if (!trouve) {
            System.out.println("😞 Perdu ! Le nombre était " + nombreSecret);
            System.out.println("Réessayez pour améliorer votre score !");
        }
        
        scanner.close();
    }
    
    public static void donnerIndice(int proposition, int nombreSecret, int tentatives) {
        int difference = Math.abs(proposition - nombreSecret);
        
        if (difference <= 3) {
            System.out.println("Vous êtes très proche !");
        } else if (difference <= 10) {
            System.out.println("Vous vous rapprochez...");
        } else if (difference <= 20) {
            System.out.println("Encore loin...");
        }
        
        // Indice supplémentaire après 3 tentatives
        if (tentatives >= 3) {
            if (nombreSecret % 2 == 0) {
                System.out.println("Indice: Le nombre est pair");
            } else {
                System.out.println("Indice: Le nombre est impair");
            }
        }
    }
    
    public static void calculerScore(int tentatives) {
        int score = 0;
        String mention = "";
        
        switch (tentatives) {
            case 1:
                score = 1000;
                mention = "EXCEPTIONNEL !";
                break;
            case 2:
                score = 800;
                mention = "EXCELLENT !";
                break;
            case 3:
                score = 600;
                mention = "Très bien !";
                break;
            case 4:
                score = 400;
                mention = "Bien !";
                break;
            case 5:
                score = 200;
                mention = "Pas mal !";
                break;
            case 6:
                score = 100;
                mention = "Juste à temps !";
                break;
            case 7:
                score = 50;
                mention = "De justesse !";
                break;
            default:
                score = 0;
                mention = "Aucun score";
        }
        
        System.out.println("Score: " + score + " points - " + mention);
        System.out.println("Trouvé en " + tentatives + " tentative" + (tentatives > 1 ? "s" : ""));
    }
}
