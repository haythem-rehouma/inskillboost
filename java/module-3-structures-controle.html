Module 3 - Structures de Contrôle

Maîtrisez le flux d'exécution de vos programmes Java

Qu'est-ce que les Structures de Contrôle ?

Les structures de contrôle sont des instructions spéciales qui permettent de contrôler l'ordre dans lequel les instructions de votre programme s'exécutent. Sans elles, votre programme ne ferait qu'exécuter les lignes une par une, de haut en bas, comme un robot qui suit une recette à la lettre.

Pourquoi avons-nous besoin des Structures de Contrôle ?

Imaginez un programme sans structures de contrôle :

// Programme "robot" sans intelligence
System.out.println("Bonjour");
System.out.println("Il fait beau");
System.out.println("Il pleut");
System.out.println("Prenez un parapluie");
System.out.println("Prenez des lunettes de soleil");
// ↑ Le programme dit TOUT, même des contradictions !

Avec les structures de contrôle, on peut créer de l'intelligence :

// Programme "intelligent" avec logique
System.out.println("Bonjour");

if (ilFaitBeau) {
    System.out.println("Il fait beau");
    System.out.println("Prenez des lunettes de soleil");
} else {
    System.out.println("Il pleut");
    System.out.println("Prenez un parapluie");
}
// ↑ Le programme s'adapte à la situation !

Les 3 Familles de Structures de Contrôle

Structures Conditionnelles

Question : "Si... alors... sinon..."

Exécuter du code seulement dans certaines conditions

  • if / else : Conditions simples
  • switch / case : Choix multiples
  • Opérateur ternaire : Condition en une ligne

Exemple : "Si il pleut, prendre un parapluie"

Structures Répétitives

Question : "Répéter tant que..."

Répéter du code plusieurs fois automatiquement

  • for : Nombre d'itérations connu
  • while : Condition testée avant
  • do-while : Condition testée après
  • for-each : Parcours de collections

Exemple : "Compter de 1 à 10"

Instructions de Saut

Question : "Sortir ou passer..."

Modifier le flux normal d'exécution

  • break : Sortir complètement
  • continue : Passer à l'itération suivante
  • return : Sortir de la méthode
  • Labels : Contrôle précis des boucles

Exemple : "Arrêter la boucle si erreur"

/**

public class OrganisationSoiree {
    public static void main(String[] args) {
        int nombreInvites = 8;
        boolean ilPleut = false;
        String[] invites = {"Alice", "Bob", "Charlie", "Diana", "Eve", "Frank", "Grace", "Henry"};
        
        System.out.println("=== ORGANISATION DE SOIRÉE ===");
        
        // 1. STRUCTURE CONDITIONNELLE : Décider du lieu
        if (nombreInvites > 10) {
            System.out.println("🏠 Soirée dans le salon (beaucoup d'invités)");
        } else if (ilPleut) {
            System.out.println("🏠 Soirée à l'intérieur (il pleut)");
        } else {
            System.out.println("🌳 Soirée dans le jardin (temps parfait)");
        }
        
        // 2. STRUCTURE RÉPÉTITIVE : Appeler tous les invités
        System.out.println("\n📞 Appel des invités:");
        for (int i = 0; i < invites.length; i++) {
            System.out.println("Appel " + (i + 1) + ": " + invites[i]);
            
            // STRUCTURE CONDITIONNELLE dans la boucle
            if (invites[i].equals("Charlie")) {
                System.out.println("   → Charlie ne peut pas venir (déjà occupé)");
                continue; // INSTRUCTION DE SAUT : passer au suivant
            }
            
            System.out.println("   → " + invites[i] + " confirmé(e) !");
        }
        
        // 3. STRUCTURE RÉPÉTITIVE : Préparer les activités
        String[] activites = {"Musique", "Jeux", "Buffet", "Photos"};
        System.out.println("\n🎉 Préparation des activités:");
        
        for (String activite : activites) { // For-each pour parcourir
            System.out.println("Préparation: " + activite);
            
            if (activite.equals("Buffet")) {
                System.out.println("   → Préparation spéciale du buffet...");
                
                // Boucle imbriquée pour les plats
                String[] plats = {"Entrées", "Plats", "Desserts"};
                for (String plat : plats) {
                    System.out.println("     • " + plat + " préparé");
                }
            }
        }
        
        System.out.println("\n🎊 Soirée prête ! Tous les invités peuvent venir.");
    }
}

Conditions if/else - Prendre des Décisions

La structure if est le moyen de faire prendre des décisions à votre programme. C'est comme dire à votre programme : "Si cette situation se présente, alors fais ceci, sinon fais cela".

Comment fonctionne une condition if ?
  1. Évaluation : Java évalue l'expression dans les parenthèses
  2. Test booléen : Le résultat doit être true ou false
  3. Exécution conditionnelle : Si true → exécute le bloc, si false → ignore
  4. Continuation : Le programme continue après le bloc if

Exemple Très Simple pour Comprendre

public class PremierIf {
    public static void main(String[] args) {
        int age = 17;
        
        System.out.println("Vous avez " + age + " ans");
        
        // Condition simple
        if (age >= 18) {
            System.out.println("Vous êtes majeur !");
            System.out.println("Vous pouvez voter");
        }
        
        System.out.println("Fin du programme");
    }
}

/*
EXÉCUTION ÉTAPE PAR ÉTAPE :
1. age = 17
2. Affiche "Vous avez 17 ans"
3. Teste la condition : 17 >= 18 ?
4. 17 >= 18 = false
5. IGNORE le bloc if (ne l'exécute pas)
6. Affiche "Fin du programme"

RÉSULTAT :
Vous avez 17 ans
Fin du programme
*/

Conditions Composées avec Opérateurs Logiques

public class ConditionsComposees {
    public static void main(String[] args) {
        int age = 25;
        double salaire = 2500;
        int experienceAnnees = 3;
        boolean aDiplome = true;
        
        System.out.println("=== ÉVALUATION CANDIDAT EMPLOI ===");
        System.out.printf("Âge: %d, Salaire souhaité: %.0f€, Expérience: %d ans, Diplôme: %s%n",
                         age, salaire, experienceAnnees, aDiplome);
        
        // Condition composée avec AND (&&)
        if (age >= 21 && age <= 35) {
            System.out.println("✅ Âge dans la tranche cible (21-35 ans)");
        } else {
            System.out.println("❌ Âge hors tranche cible");
        }
        
        // Condition composée avec OR (||)
        if (experienceAnnees >= 5 || aDiplome) {
            System.out.println("✅ Profil qualifié (expérience OU diplôme)");
        } else {
            System.out.println("❌ Profil insuffisant (pas d'expérience ET pas de diplôme)");
        }
        
        // Condition complexe avec parenthèses
        if ((experienceAnnees >= 2 && aDiplome) || (experienceAnnees >= 5)) {
            System.out.println("✅ Critères techniques remplis");
            
            // Condition imbriquée pour le salaire
            if (salaire <= 3000) {
                System.out.println("✅ Prétentions salariales acceptables");
                System.out.println("🎉 CANDIDAT RETENU !");
            } else {
                System.out.println("❌ Prétentions salariales trop élevées");
                System.out.println("💰 Négociation nécessaire");
            }
        } else {
            System.out.println("❌ Critères techniques non remplis");
            System.out.println("📚 Formation recommandée");
        }
        
        // Condition avec NOT (!)
        if (!aDiplome && experienceAnnees < 2) {
            System.out.println("⚠️  Profil débutant détecté");
            System.out.println("🎓 Stage ou formation en alternance recommandé");
        }
    }
}

/*
EXPLICATION DES OPÉRATEURS LOGIQUES :

&& (AND) : Les DEUX conditions doivent être vraies
age >= 21 && age <= 35
↑           ↑
true    ET  true  = true
true    ET  false = false
false   ET  true  = false
false   ET  false = false

|| (OR) : AU MOINS UNE condition doit être vraie
experienceAnnees >= 5 || aDiplome
↑                        ↑
true     OU  true  = true
true     OU  false = true
false    OU  true  = true
false    OU  false = false

! (NOT) : Inverse le résultat
!aDiplome
↑
Si aDiplome = true  → !aDiplome = false
Si aDiplome = false → !aDiplome = true
*/

Exemple Concret : Système de Connexion

import java.util.Scanner;

public class SystemeConnexion {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        // Données "en dur" pour la démonstration
        String utilisateurCorrect = "admin";
        String motDePasseCorrect = "java123";
        int tentativesMax = 3;
        int tentatives = 0;
        
        System.out.println("=== SYSTÈME DE CONNEXION ===");
        
        boolean connecte = false;
        
        // Boucle pour les tentatives de connexion
        while (tentatives < tentativesMax && !connecte) {
            tentatives++;
            System.out.println("\n--- Tentative " + tentatives + "/" + tentativesMax + " ---");
            
            System.out.print("Nom d'utilisateur: ");
            String utilisateur = scanner.nextLine();
            
            System.out.print("Mot de passe: ");
            String motDePasse = scanner.nextLine();
            
            // Vérification des identifiants avec conditions composées
            if (utilisateur.equals(utilisateurCorrect) && motDePasse.equals(motDePasseCorrect)) {
                System.out.println("✅ CONNEXION RÉUSSIE !");
                System.out.println("Bienvenue, " + utilisateur + " !");
                connecte = true;
                
                // Menu après connexion
                afficherMenuPrincipal();
                
            } else if (utilisateur.equals(utilisateurCorrect)) {
                System.out.println("❌ Nom d'utilisateur correct, mais mot de passe incorrect");
                System.out.println("💡 Vérifiez les majuscules/minuscules");
                
            } else if (motDePasse.equals(motDePasseCorrect)) {
                System.out.println("❌ Mot de passe correct, mais nom d'utilisateur incorrect");
                System.out.println("💡 Vérifiez l'orthographe du nom d'utilisateur");
                
            } else {
                System.out.println("❌ Nom d'utilisateur ET mot de passe incorrects");
                System.out.println("💡 Vérifiez vos identifiants");
            }
            
            // Avertissement selon le nombre de tentatives restantes
            if (!connecte && tentatives < tentativesMax) {
                int restantes = tentativesMax - tentatives;
                if (restantes == 1) {
                    System.out.println("⚠️  ATTENTION : Dernière tentative !");
                } else {
                    System.out.println("Il vous reste " + restantes + " tentative(s)");
                }
            }
        }
        
        // Vérification finale
        if (!connecte) {
            System.out.println("\n🚫 COMPTE BLOQUÉ !");
            System.out.println("Trop de tentatives échouées");
            System.out.println("Contactez l'administrateur");
        }
        
        scanner.close();
    }
    
    public static void afficherMenuPrincipal() {
        System.out.println("\n=== MENU PRINCIPAL ===");
        System.out.println("1. Consulter le profil");
        System.out.println("2. Modifier les paramètres");
        System.out.println("3. Se déconnecter");
        System.out.println("Vous êtes maintenant connecté au système !");
    }
}

/*
ANALYSE DU CODE :

1. BOUCLE while avec condition composée :
   while (tentatives < tentativesMax && !connecte)
   Continue TANT QUE : pas trop de tentatives ET pas encore connecté

2. CONDITIONS IMBRIQUÉES pour analyser les erreurs :
   - if (user OK && password OK) → succès
   - else if (user OK) → password incorrect
   - else if (password OK) → user incorrect  
   - else → tout incorrect

3. LOGIQUE MÉTIER avec conditions :
   - Compteur de tentatives
   - Messages d'aide selon l'erreur
   - Blocage après trop de tentatives

C'est un exemple RÉEL d'utilisation des structures de contrôle !
*/

Switch/Case

La structure switch permet de comparer une variable avec plusieurs valeurs possibles. Plus lisible que de multiples if/else pour les comparaisons d'égalité.

Syntaxe Classique

public class ExempleSwitch {
    public static void main(String[] args) {
        int jourSemaine = 3;
        String nomJour;
        
        switch (jourSemaine) {
            case 1:
                nomJour = "Lundi";
                break;
            case 2:
                nomJour = "Mardi";
                break;
            case 3:
                nomJour = "Mercredi";
                break;
            case 4:
                nomJour = "Jeudi";
                break;
            case 5:
                nomJour = "Vendredi";
                break;
            case 6:
                nomJour = "Samedi";
                break;
            case 7:
                nomJour = "Dimanche";
                break;
            default:
                nomJour = "Jour invalide";
                break;
        }
        
        System.out.println("Jour " + jourSemaine + ": " + nomJour);
    }
}

Switch Expression (Java 14+)

public class SwitchModerne {
    public static void main(String[] args) {
        int jourSemaine = 3;
        
        // Switch expression (plus concis)
        String nomJour = switch (jourSemaine) {
            case 1 -> "Lundi";
            case 2 -> "Mardi";
            case 3 -> "Mercredi";
            case 4 -> "Jeudi";
            case 5 -> "Vendredi";
            case 6 -> "Samedi";
            case 7 -> "Dimanche";
            default -> "Jour invalide";
        };
        
        System.out.println("Jour: " + nomJour);
        
        // Switch avec yield pour logique complexe
        String typeJour = switch (jourSemaine) {
            case 1, 2, 3, 4, 5 -> {
                System.out.println("C'est un jour de semaine");
                yield "Jour ouvrable";
            }
            case 6, 7 -> {
                System.out.println("C'est le week-end");
                yield "Week-end";
            }
            default -> {
                System.out.println("Jour invalide");
                yield "Erreur";
            }
        };
        
        System.out.println("Type: " + typeJour);
    }
}
Bonnes Pratiques Switch :
  • Toujours inclure un default
  • Utiliser break pour éviter le fall-through
  • Préférer switch expression quand possible (Java 14+)
  • Grouper les cas similaires avec des virgules

Boucle for - Répéter un Nombre Précis de Fois

La boucle for est parfaite quand vous savez exactement combien de fois vous voulez répéter quelque chose. C'est comme dire : "Fais cette action 10 fois" ou "Parcours tous les éléments de cette liste".

Quand utiliser une boucle for ?
  • Comptage : "Compter de 1 à 100"
  • Parcours de tableau : "Afficher tous les éléments"
  • Répétition précise : "Demander 5 notes à l'utilisateur"
  • Calculs itératifs : "Calculer la factorielle de 10"
  • Génération de données : "Créer 50 nombres aléatoires"

Anatomie d'une Boucle for

// Syntaxe détaillée avec explications
for (int i = 1; i <= 5; i++) {
//   ↑       ↑      ↑
//   │       │      └── 3. INCRÉMENTATION : Exécutée APRÈS chaque tour
//   │       └────────── 2. CONDITION : Testée AVANT chaque tour
//   └────────────────── 1. INITIALISATION : Exécutée UNE SEULE FOIS au début
    
    System.out.println("Tour numéro: " + i);
}

/*
EXÉCUTION ÉTAPE PAR ÉTAPE :

DÉBUT:
1. Initialisation : i = 1

TOUR 1:
2. Test condition : 1 <= 5 ? → true
3. Exécute le corps : "Tour numéro: 1"
4. Incrémentation : i++ → i = 2

TOUR 2:
2. Test condition : 2 <= 5 ? → true
3. Exécute le corps : "Tour numéro: 2"
4. Incrémentation : i++ → i = 3

TOUR 3:
2. Test condition : 3 <= 5 ? → true
3. Exécute le corps : "Tour numéro: 3"
4. Incrémentation : i++ → i = 4

Boucles while et do-while - Répéter Tant Que...

Les boucles while et do-while répètent du code tant qu'une condition reste vraie. Contrairement à for, vous ne savez pas à l'avance combien de fois la boucle va s'exécuter.

Différence Fondamentale : for vs while
  • for : "Répète exactement 10 fois" (nombre connu)
  • while : "Répète jusqu'à ce que l'utilisateur tape 'stop'" (nombre inconnu)
  • for : Idéal pour parcours, comptages
  • while : Idéal pour interactions utilisateur, conditions complexes

Boucle while (Test AVANT)

La condition est testée avant chaque exécution

int compteur = 1;

while (compteur <= 5) {
    System.out.println("Compteur: " + compteur);
    compteur++; // IMPORTANT : modifier la variable de condition !
}

/*
ÉTAPES :
1. Test : 1 <= 5 ? → true → exécute
2. Test : 2 <= 5 ? → true → exécute  
3. Test : 3 <= 5 ? → true → exécute
4. Test : 4 <= 5 ? → true → exécute
5. Test : 5 <= 5 ? → true → exécute
6. Test : 6 <= 5 ? → false → STOP

⚠️  DANGER : Si vous oubliez compteur++, 
boucle infinie car compteur reste toujours 1 !
*/

Boucle do-while (Test APRÈS)

La condition est testée après chaque exécution

int compteur = 1;

do {
    System.out.println("Compteur: " + compteur);
    compteur++;
} while (compteur <= 5);

/*
ÉTAPES :
1. Exécute → puis test : 2 <= 5 ? → true
2. Exécute → puis test : 3 <= 5 ? → true
3. Exécute → puis test : 4 <= 5 ? → true
4. Exécute → puis test : 5 <= 5 ? → true
5. Exécute → puis test : 6 <= 5 ? → false → STOP

✅ GARANTIE : S'exécute AU MOINS une fois,
même si la condition est fausse dès le début
*/

            for (int j = 1; j <= taille - i; j++) {
                System.out.print(" ");

                System.out.print("*");

                System.out.print(" ");
            }
            for (int j = 1; j <= 2 * i - 1; j++) {
                System.out.print("*");

        System.out.println("\n4. SAPIN DE NOËL:");

Exercice 3 : Algorithmes de Recherche et Tri

Objectif : Implémenter des algorithmes fondamentaux avec les boucles.

Algorithmes à implémenter :

  • Recherche linéaire dans un tableau
  • Recherche du minimum et maximum
  • Tri à bulles (Bubble Sort)
  • Calcul de statistiques (moyenne, médiane)
Solution Complète :
public class AlgorithmesFondamentaux {
    public static void main(String[] args) {
        int[] tableau = {64, 34, 25, 12, 22, 11, 90, 88, 76, 50, 42};
        
        System.out.println("=== ALGORITHMES FONDAMENTAUX ===");
        System.out.print("Tableau initial: ");
        afficherTableau(tableau);
        System.out.println();
        
        // 1. Recherche linéaire
        int elementRecherche = 22;
        int position = rechercheLineaire(tableau, elementRecherche);
        if (position != -1) {
            System.out.println("Élément " + elementRecherche + " trouvé à l'index " + position);
        } else {
            System.out.println("Élément " + elementRecherche + " non trouvé");
        }
        
        // 2. Recherche min/max
        int[] minMax = trouverMinMax(tableau);
        System.out.println("Minimum: " + minMax[0] + ", Maximum: " + minMax[1]);
        
        // 3. Calcul statistiques
        double[] stats = calculerStatistiques(tableau);
        System.out.printf("Moyenne: %.2f, Médiane: %.2f%n", stats[0], stats[1]);
        
        // 4. Tri à bulles
        int[] tableauTrie = tableau.clone(); // Copie pour ne pas modifier l'original
        triBulles(tableauTrie);
        System.out.print("Tableau trié: ");
        afficherTableau(tableauTrie);
        System.out.println();
    }
    
    public static int rechercheLineaire(int[] tableau, int element) {
        for (int i = 0; i < tableau.length; i++) {
            if (tableau[i] == element) {
                return i; // Retourne l'index de l'élément trouvé
            }
        }
        return -1; // Élément non trouvé
    }
    
    public static int[] trouverMinMax(int[] tableau) {
        if (tableau.length == 0) {
            return new int[]{0, 0};
        }
        
        int min = tableau[0];
        int max = tableau[0];
        
        for (int i = 1; i < tableau.length; i++) {
            if (tableau[i] < min) {
                min = tableau[i];
            }
            if (tableau[i] > max) {
                max = tableau[i];
            }
        }
        
        return new int[]{min, max};
    }
    
    public static double[] calculerStatistiques(int[] tableau) {
        if (tableau.length == 0) {
            return new double[]{0.0, 0.0};
        }
        
        // Calcul de la moyenne
        double somme = 0;
        for (int valeur : tableau) {
            somme += valeur;
        }
        double moyenne = somme / tableau.length;
        
        // Calcul de la médiane (nécessite un tri)
        int[] tableauTrie = tableau.clone();
        triBulles(tableauTrie);
        
        double mediane;
        int milieu = tableauTrie.length / 2;
        if (tableauTrie.length % 2 == 0) {
            mediane = (tableauTrie[milieu - 1] + tableauTrie[milieu]) / 2.0;
        } else {
            mediane = tableauTrie[milieu];
        }
        
        return new double[]{moyenne, mediane};
    }
    
    public static void triBulles(int[] tableau) {
        int n = tableau.length;
        boolean echange;
        
        for (int i = 0; i < n - 1; i++) {
            echange = false;
            
            for (int j = 0; j < n - i - 1; j++) {
                if (tableau[j] > tableau[j + 1]) {
                    // Échange des éléments
                    int temp = tableau[j];
                    tableau[j] = tableau[j + 1];
                    tableau[j + 1] = temp;
                    echange = true;
                }
            }
            
            // Si aucun échange, le tableau est trié
            if (!echange) {
                break;
            }
        }
    }
    
    public static void afficherTableau(int[] tableau) {
        System.out.print("[");
        for (int i = 0; i < tableau.length; i++) {
            System.out.print(tableau[i]);
            if (i < tableau.length - 1) {
                System.out.print(", ");
            }
        }
        System.out.print("]");
    }
}

    public static void operationsBase() {
        System.out.print("Premier nombre: ");
        double a = scanner.nextDouble();
        System.out.print("Opérateur (+, -, *, /): ");
        String op = scanner.next();
        System.out.print("Deuxième nombre: ");
        double b = scanner.nextDouble();
        
        double resultat = 0;
        boolean operationValide = true;
        String calcul = a + " " + op + " " + b;
        
        switch (op) {
            case "+":
                resultat = a + b;
                break;
            case "-":
                resultat = a - b;
                break;
            case "*":
                resultat = a * b;
                break;
            case "/":
                if (b != 0) {
                    resultat = a / b;
                } else {
                    System.out.println("Erreur: Division par zéro impossible");
                    operationValide = false;
                }
                break;
            default:
                System.out.println("Opérateur invalide");
                operationValide = false;

            System.out.printf("Résultat: %.4f%n", resultat);

    public static void calculerPuissance() {
        System.out.print("Base: ");
        double base = scanner.nextDouble();
        System.out.print("Exposant: ");
        int exposant = scanner.nextInt();
        
        double resultat = 1;
        
        // Calcul de la puissance avec une boucle
        if (exposant >= 0) {
            for (int i = 0; i < exposant; i++) {
                resultat *= base;
            }
        } else {
            for (int i = 0; i > exposant; i--) {
                resultat /= base;
            }
        }
        
        System.out.printf("%.2f^%d = %.6f%n", base, exposant, resultat);
        historique.add(base + "^" + exposant + " = " + resultat);
    }
    
    public static void calculerRacine() {
        System.out.print("Nombre: ");
        double nombre = scanner.nextDouble();
        
        if (nombre < 0) {
            System.out.println("Erreur: Racine carrée d'un nombre négatif");
            return;
        }
        
        // Méthode de Newton pour calculer la racine carrée
        double resultat = nombre;
        double precision = 0.000001;
        
        while (Math.abs(resultat * resultat - nombre) > precision) {
            resultat = (resultat + nombre / resultat) / 2;
        }
        
        System.out.printf("√%.2f = %.6f%n", nombre, resultat);
        historique.add("√" + nombre + " = " + resultat);
    }
    
    public static void calculerFactorielle() {
        System.out.print("Nombre entier positif: ");

        if (n > 20) {
            System.out.println("Attention: Résultat très grand pour n > 20");
        }
        
        long resultat = 1;
        
        for (int i = 1; i <= n; i++) {
            resultat *= i;
        }
        
        System.out.println(n + "! = " + resultat);
        historique.add(n + "! = " + resultat);
        
        // Affichage du calcul détaillé pour les petits nombres
        if (n <= 10) {
            System.out.print("Calcul: ");
            for (int i = 1; i <= n; i++) {
                System.out.print(i);
                if (i < n) System.out.print(" × ");
            }
            System.out.println(" = " + resultat);
        }
    }
    
    public static void afficherHistorique() {
        System.out.println("\n=== HISTORIQUE DES CALCULS ===");
        if (historique.isEmpty()) {
            System.out.println("Aucun calcul dans l'historique");
        } else {
            for (int i = 0; i < historique.size(); i++) {
                System.out.println((i + 1) + ". " + historique.get(i));
            }
        }
    }
    
    public static void viderHistorique() {
        historique.clear();
        System.out.println("Historique vidé");
    }
