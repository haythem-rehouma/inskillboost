Module 1 - Introduction à Java et Environnement

Découverte de Java, installation et premier programme

Introduction à Java

Java est un langage de programmation orienté objet, développé par Sun Microsystems (maintenant Oracle) en 1995. C'est l'un des langages les plus populaires au monde, utilisé pour développer des applications d'entreprise, des applications web, des applications mobiles Android, et bien plus encore.

Pourquoi apprendre Java ?
  • Popularité : Utilisé par des millions de développeurs dans le monde
  • Portabilité : "Write Once, Run Anywhere" (WORA)
  • Robustesse : Gestion automatique de la mémoire et typage fort
  • Écosystème : Frameworks riches (Spring, Hibernate, etc.)
  • Opportunités : Forte demande sur le marché du travail

Histoire et Évolution de Java

Chronologie Importante

  • 1991 : Projet Green (James Gosling)
  • 1995 : Première version publique
  • 1996 : JDK 1.0 - Première version stable
  • 1998 : Java 2 (J2SE, J2EE, J2ME)
  • 2004 : Java 5 - Génériques et annotations
  • 2014 : Java 8 - Lambda et Stream API
  • 2017 : Java 9 - Modules (Project Jigsaw)
  • 2023 : Java 21 - Version LTS actuelle

Versions LTS (Long Term Support)

Versions recommandées pour la production :

  • Java 8 (2014) - Support jusqu'en 2030
  • Java 11 (2018) - Support jusqu'en 2026
  • Java 17 (2021) - Support jusqu'en 2029
  • Java 21 (2023) - Support jusqu'en 2031

Caractéristiques Clés de Java

Orienté Objet

Tout en Java est objet (sauf les types primitifs). Support complet de l'encapsulation, l'héritage et le polymorphisme.

Indépendant de la Plateforme

Le bytecode Java s'exécute sur n'importe quelle JVM, quel que soit le système d'exploitation.

Gestion Automatique de la Mémoire

Le Garbage Collector libère automatiquement la mémoire des objets non utilisés.

Multithreading

Support natif de la programmation concurrente avec les threads.

Premier Programme Java

Créons notre premier programme Java - le traditionnel "Hello World" :

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
        System.out.println("Bienvenue dans la formation Java !");
    }
}

Analyse du Code

  • public class HelloWorld : Déclaration d'une classe publique
  • public static void main : Point d'entrée du programme
  • String[] args : Paramètres de ligne de commande
  • System.out.println() : Affichage dans la console

Compilation et Exécution

Processus de Compilation Java

1. Code Source (.java)

Fichier texte contenant le code Java lisible par les humains.

2. Bytecode (.class)

Code intermédiaire généré par le compilateur javac.

Commandes de Compilation et d'Exécution

# Compilation du fichier source
javac HelloWorld.java

# Cela génère HelloWorld.class

# Exécution du programme
java HelloWorld

# Sortie attendue :
# Hello, World!
# Bienvenue dans la formation Java !
Architecture JVM

La Java Virtual Machine (JVM) exécute le bytecode :

  • Class Loader : Charge les classes en mémoire
  • Bytecode Verifier : Vérifie la sécurité du code
  • Execution Engine : Exécute le bytecode
  • Garbage Collector : Gère la mémoire automatiquement

Exercices Pratiques

Exercice 1 : Premier Programme Personnalisé

Objectif : Créer un programme qui affiche vos informations personnelles.

Instructions :

  1. Créez une classe MonProfil
  2. Affichez votre nom, âge et ville
  3. Utilisez plusieurs System.out.println()
  4. Compilez et exécutez le programme
Solution :
public class MonProfil {
    public static void main(String[] args) {
        System.out.println("=== MON PROFIL ===");
        System.out.println("Nom : Jean Dupont");
        System.out.println("Age : 25 ans");
        System.out.println("Ville : Paris");
        System.out.println("Passion : Programmation Java");
        System.out.println("==================");
    }
}
Exercice 2 : Calculatrice Simple

Objectif : Créer un programme qui effectue des calculs simples.

Instructions :

  1. Créez une classe Calculatrice
  2. Déclarez deux variables entières
  3. Effectuez les 4 opérations de base
  4. Affichez les résultats avec des messages clairs
Solution :
public class Calculatrice {
    public static void main(String[] args) {
        int nombre1 = 15;
        int nombre2 = 4;
        
        System.out.println("=== CALCULATRICE SIMPLE ===");
        System.out.println("Nombre 1 : " + nombre1);
        System.out.println("Nombre 2 : " + nombre2);
        System.out.println();
        
        System.out.println("Addition : " + nombre1 + " + " + nombre2 + " = " + (nombre1 + nombre2));
        System.out.println("Soustraction : " + nombre1 + " - " + nombre2 + " = " + (nombre1 - nombre2));
        System.out.println("Multiplication : " + nombre1 + " * " + nombre2 + " = " + (nombre1 * nombre2));
        System.out.println("Division : " + nombre1 + " / " + nombre2 + " = " + (nombre1 / nombre2));
        System.out.println("Modulo : " + nombre1 + " % " + nombre2 + " = " + (nombre1 % nombre2));
    }
}

Exercice 4 : Défi - Programme Interactif

Objectif : Créer un programme qui utilise Scanner pour l'interaction utilisateur.

Instructions :

  1. Importez la classe Scanner
  2. Demandez le nom de l'utilisateur
  3. Demandez son âge
  4. Affichez un message personnalisé
Solution :
import java.util.Scanner;

public class ProgrammeInteractif {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("=== PROGRAMME INTERACTIF ===");
        
        System.out.print("Entrez votre nom : ");
        String nom = scanner.nextLine();
        
        System.out.print("Entrez votre âge : ");
        int age = scanner.nextInt();
        
        System.out.println();
        System.out.println("Bonjour " + nom + " !");
        System.out.println("Vous avez " + age + " ans.");
        
        if (age >= 18) {
            System.out.println("Vous êtes majeur.");
        } else {
            System.out.println("Vous êtes mineur.");
        }
        
        scanner.close();
    }
}
Exercice 5 : Projet - Générateur de Cartes de Visite

Objectif : Créer un générateur de cartes de visite en mode console.

Spécifications :

  • Demander : nom, prénom, métier, email, téléphone
  • Générer une carte de visite ASCII art
  • Utiliser des bordures et formatage
  • Gérer les cas d'erreur (champs vides)
Solution Complète :
import java.util.Scanner;

public class GenerateurCarteVisite {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("=== GÉNÉRATEUR DE CARTE DE VISITE ===");
        System.out.println();
        
        // Collecte des informations
        System.out.print("Prénom : ");
        String prenom = scanner.nextLine();
        
        System.out.print("Nom : ");
        String nom = scanner.nextLine();
        
        System.out.print("Métier : ");
        String metier = scanner.nextLine();
        
        System.out.print("Email : ");
        String email = scanner.nextLine();
        
        System.out.print("Téléphone : ");
        String telephone = scanner.nextLine();
        
        // Génération de la carte
        System.out.println();
        System.out.println("╔══════════════════════════════════════╗");
        System.out.println("║            CARTE DE VISITE           ║");
        System.out.println("╠══════════════════════════════════════╣");
        System.out.println("║ " + String.format("%-36s", prenom + " " + nom.toUpperCase()) + " ║");
        System.out.println("║ " + String.format("%-36s", metier) + " ║");
        System.out.println("║                                      ║");
        System.out.println("║ Email: " + String.format("%-28s", email) + " ║");
        System.out.println("║ Tel:   " + String.format("%-28s", telephone) + " ║");
        System.out.println("╚══════════════════════════════════════╝");
        
        scanner.close();
    }
}
